# Introduction 
Graphex plugin enabling esxi utils python tools library as nodes.

# Installation
Run 'make all' to install this plugin.

# Execution
The code in this repo is not meant to be executed. It is an interface between the python library 'esxi_utils' and the
'graphex' python module. Any code here is meant to be loaded via the 'graphex' application.
